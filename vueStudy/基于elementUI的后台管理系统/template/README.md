## 使用

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run e2e tests
npm run e2e

# run unit tests
npm run unit

# run all tests
run test
```

## 结构

```
| - src
  | - api           定义请求接口
  | - components    存放定义的组件
  | - store         vuex
  | - views         存放路由页面
  | - nav-config.js 路由配置
| - static          静态资源  
```
